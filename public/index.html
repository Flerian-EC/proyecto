El código del color

Para entender el código de los colores es necesario conocer el modelo RGB.

RGB

Este modelo consiste en mezclar 3 colores primarios de luz (rojo, verde, azul) para obtener una gama de colores distintos, este modelo fue desarrollado por la “comisión internacional de iluminación”

Aquí nos podríamos preguntar ¿Para qué es necesario este modelo?. Es necesario para tener un consenso. Se decidió establecer un código de colores universal de modo que los colores RBG se representan como una combinación de números y letras en las que un ordenador puede distinguir hasta unos 16 millones de colores diferentes. Quien se encarga de pintar estos colores en pantalla son los pixeles.

Pixel

El pixel es la unidad mas pequeña de una imagen reproducida digitalmente. Actualmente las pantallas tienen cientos o miles de pixeles, dependiendo de la resolución del dispositivo. Para que un pixel pueda pintar un color especifico, se compone de 3 subpíxeles en colores rojo verde y azul (modelo RGB), la combinación de estos 3 subpíxeles en tonos diferentes de RGB genera el color que pintara ese pixel.

Codificación hexadecimal de color

Usando la notación hexadecimal podemos expresar sencillamente un color concreto de la escala RGB, este sistema utiliza la combinación de tres conjuntos de 2 dígitos entre “00 “y “FF” para expresar las diferentes intensidades de los colores RGB (rojo, verde, azul) y así generar un color.

También se puede usar la notación decimal para representar un color RGB, esta notación usa tres conjuntos de números entre 0 y 255 para expresar las diferentes intensidades de los colores RGB (rojo, verde, azul) y así generar un color.

En el siguiente ejemplo se puede ver de forma mas grafica estos conjuntos y también nos damos cuenta que todos los códigos de color inician con el carácter “#”.

Uso del código de color

Para que todo esto nos pueda ser más útil es necesario alguna pista que nos ayude a visualizar los colores a partir de sus códigos, aunque sea de modo aproximado. Tomando el siguiente ejemplo de código de color

#27ECC0

Ya sabemos que el código se divide en 3 conjuntos. El primer carácter de cada conjunto es el que más influye en la ecuación, así que para hacer una lectura a ojo es útil fijarnos en este primer carácter. Así tenemos que en este caso, rojo hay poca intensidad (2), mientras que de verde hay mucha intensidad (E), y de azul también hay bastante pero no tanta como de verde (C).Esta lectura nos da la idea de que el color tenderá hacia tonos verdes intensos y azulados intensos.
